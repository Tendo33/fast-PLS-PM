# 分析报告：Urolithin A 的关系分析

## 方法
模型：结构方程模型 (SEM)
库：`semopy` (Python)
注：最初尝试使用 `plspm` 库，但遇到内部错误。因此改用 SEM 来估计路径关系，对于直接效应的解释是等效的。

## 模型结构
- **外生变量 (输入):** Urolithin A 剂量 (`UroA`)
- **内生变量 (结果):** 
    - 产蛋性能 (`Egg`)
    - 肝脏重量/指数 (`LiverW`)
    - 腹脂重量 (`FatW`)
    - 血清脂质代谢 (`SerumLip`)
    - 肝脏脂质代谢 (`LiverLip`)
    - 肝损伤 (`LiverInj`)

## 结果：路径系数 (Urolithin A 的影响)

| 结果变量 | 估计值 (关系强度) | P值 | 显著性 |
| :--- | :--- | :--- | :--- |
| **肝脏重量** (`LiverW`) | **-7.42** | **0.0001** | ***显著 (强负相关)** |
| **腹脂重量** (`FatW`) | **-23.14** | **0.0043** | **显著 (强负相关)** |
| **血清脂质** (`SerumLip`) | **-3.68** | **0.0007** | ***显著 (负相关)** |
| **产蛋性能** (`Egg`) | -1.81 | 0.576 | 不显著 |
| **肝脏脂质** (`LiverLip`) | 0.78 | 0.569 | 不显著 |
| **肝损伤** (`LiverInj`) | -1.01 | 0.149 | 不显著 |

## 解释
Urolithin A 的添加对以下指标有统计学上显著的 **负向影响**（即剂量越高，指标值越低）：
- **肝脏重量**：较高剂量与较低的肝脏重量/指数相关。
- **腹脂重量**：较高剂量与显著较低的腹脂重量相关。
- **血清脂质**：较高剂量与较低的血清脂质水平相关。

在本数据集/模型中，未观察到对产蛋性能、肝脏脂质水平或肝损伤标志物的显著影响。
